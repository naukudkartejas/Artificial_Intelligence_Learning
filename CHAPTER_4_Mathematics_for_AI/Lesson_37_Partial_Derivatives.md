# Lesson 37: Partial Derivatives

> Goal: Learn how to differentiate functions with multiple variables.

Neural networks have millions of parameters.

We compute the derivative with respect to **one parameter at a time**.

``` text
Loss(x,y,z)

↓

∂Loss/∂x
∂Loss/∂y
∂Loss/∂z
```

Applications: - Neural network training - Optimization

Summary: Partial derivatives tell how one variable affects the output
while others stay fixed.
