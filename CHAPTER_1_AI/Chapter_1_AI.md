# Chapter 1: Artificial Intelligence (AI)

> **Goal:** Understand what AI is, why it was created, where it came
> from, and how it is used in the real world.

------------------------------------------------------------------------

# 1. Before Learning AI

Forget these common misconceptions:

-   AI = ChatGPT ❌
-   AI = Robots ❌
-   AI = Humans replaced ❌

These are **applications** of AI, not AI itself.

------------------------------------------------------------------------

# 2. The World Before AI

Imagine it's **1950**.

Computers could perform calculations:

``` text
2 + 3 = 5
100 × 25 = 2500
```

They were excellent calculators but terrible at understanding the world.

Suppose you show a computer a cat.

To us:

``` text
 /\_/\\
( o.o )
 > ^ <
```

To the computer:

``` text
101010001101011
010101011100101
111010100101101
```

A computer only sees numbers.

------------------------------------------------------------------------

# 3. Traditional Programming

The programmer writes rules.

``` text
IF age >= 18
    Eligible
ELSE
    Not Eligible
```

ATM example:

``` text
IF balance >= withdrawal
    Allow
ELSE
    Reject
```

Architecture:

``` text
Programmer
    │
    ▼
Writes Rules
    │
    ▼
Computer
    │
    ▼
Answer
```

Works perfectly when the rules are obvious.

------------------------------------------------------------------------

# 4. The Big Problem

Now write rules for identifying a dog.

Rule 1

``` text
Has four legs
```

Problem:

Tables also have four legs.

Rule 2

``` text
Has fur
```

Problem:

Cats.

Rule 3

``` text
Has a tail
```

Problem:

Some dogs don't.

Millions of possible situations make manual rule writing impossible.

------------------------------------------------------------------------

# 5. Birth of Artificial Intelligence

Scientists asked:

> Instead of writing every rule manually...

> Can computers learn the rules themselves?

That question gave birth to **Artificial Intelligence**.

------------------------------------------------------------------------

# 6. Definition

**Artificial Intelligence (AI)**

> The field of making machines perform tasks that normally require human
> intelligence.

Examples include:

-   Face recognition
-   Speech recognition
-   Language translation
-   Recommendation systems
-   Medical diagnosis
-   Game playing
-   Decision making

------------------------------------------------------------------------

# 7. Why is it Called Artificial Intelligence?

Artificial = Human-made

Intelligence = Ability to learn, reason, solve problems and adapt.

``` text
Artificial
      +
Intelligence
      =
Human-made Intelligence
```

------------------------------------------------------------------------

# 8. Does AI Really Think?

No.

AI mainly learns statistical patterns from data.

Human:

Sees many dogs during life.

↓

Learns what a dog looks like.

AI:

Sees thousands or millions of labeled dog images.

↓

Learns patterns.

↓

Predicts whether a new image contains a dog.

------------------------------------------------------------------------

# 9. Parent and Child Analogy

``` text
Parent

↓

Shows Apple

↓

"This is an apple."

↓

Shows another apple

↓

"This is also an apple."

↓

Shows thousands more
```

AI learns similarly, but from much larger datasets.

# 13. Interview Questions

## Basic

**Q:** What is Artificial Intelligence?

**Answer:**

Artificial Intelligence is the field of computer science that develops
systems capable of performing tasks that normally require human
intelligence, such as learning, reasoning, perception and language
understanding.

------------------------------------------------------------------------

## Intermediate

**Q:** Why was AI invented?

**Answer:**

Because many real-world problems cannot be solved effectively using
manually written rules. AI allows computers to learn patterns from data.

------------------------------------------------------------------------

## Advanced

**Q:** Is every intelligent-looking program AI?

**Answer:**

No.

Programs that only follow fixed rules are traditional software.

AI systems can learn, predict, adapt or reason from data.

------------------------------------------------------------------------

# 14. Common Misconceptions

❌ AI is only robots.

✅ Robots may use AI, but AI also powers search engines, recommendation
systems and medical software.

------------------------------------------------------------------------

❌ AI is only ChatGPT.

✅ ChatGPT is one application built using Large Language Models, which
are one branch of AI.

------------------------------------------------------------------------

❌ AI understands like humans.

✅ AI primarily identifies statistical patterns. Human understanding is
much richer and grounded in lived experience.

# Key Takeaways

-   AI exists because writing rules for every real-world problem is
    impossible.
-   AI enables computers to learn patterns instead of relying only on
    manually coded rules.
-   AI is an umbrella field that includes Machine Learning, Deep
    Learning, NLP, Computer Vision, Robotics, Transformers, Large
    Language Models and Generative AI.
-   AI is used in nearly every major industry today.
